什么是svg

Scalable Vector Graphics (SVG)可缩放的矢量图形，应用几个定义wiki

  • SVG 指可伸缩矢量图形 (Scalable Vector Graphics)
  • SVG 用来定义用于网络的基于矢量的图形
  • SVG 使用 XML 格式定义图形
  • SVG 图像在放大或改变尺寸的情况下其图形质量不会有所损失
  • SVG 是万维网联盟的标准
  • SVG 与诸如 DOM和 XSL 之类的W3C标准是一个整体
